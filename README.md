# Portifólio_Leaxdro
Este projeto consiste no desenvolvimento de um website de portfólio pessoal, criado como parte da formação acadêmica na São Paulo Tech School. O objetivo principal é integrar, na prática, os conhecimentos adquiridos ao longo do curso, abrangendo áreas como desenvolvimento web, banco de dados, arquitetura computacional e análise de dados.

O sistema apresenta informações sobre o autor, incluindo projetos, interesses, conquistas e formas de contato, por meio de uma interface moderna em tema dark. Além disso, o projeto conta com funcionalidades como coleta e armazenamento de dados, organização de informações e um dashboard interativo para visualização de indicadores.
